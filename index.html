Q1
LONG
How does a CPU work? Explain ALU, Control Unit, Registers, and the Fetch-Execute Cycle.
🔴 ★★★★★ MUST DO FIRST 📅 7 sources: 2025PE · 2025SG · 2024 · 2023 · 2022 · DJ Prelim · BIEK Model
▼
✍ Write Exactly Like This in Your Answer Paper
📌 Introduction
CPU stands for Central Processing Unit. It is the most important component of a computer — called the "brain of the computer." All processing, calculations, and decision-making happen inside the CPU.
🔧 Main Parts of CPU
1. ALU — Arithmetic Logic Unit: The calculating part. Performs two types of operations:
Arithmetic: addition, subtraction, multiplication, division
Logical: comparisons — greater than, less than, AND, OR, NOT
2. Control Unit (CU): The coordinator/manager. Does NOT process data itself. Instead it:
Fetches instructions from RAM
Decodes (understands) those instructions
Sends control signals to all other parts
Controls flow of data between CPU, memory, and I/O devices
3. Registers — extremely fast, tiny storage inside CPU. Key registers:
PC (Program Counter): Holds address of the NEXT instruction
IR (Instruction Register): Holds the CURRENT instruction being executed
ACC (Accumulator): Stores results of ALU operations
MAR (Memory Address Register): Holds address of memory location being accessed
MBR (Memory Buffer Register): Temporarily holds data going to/from memory
🔄 Fetch-Execute Cycle
The CPU continuously repeats a 3-step cycle:
Step 1 — FETCH: Control Unit reads address from Program Counter Fetches next instruction from RAM → places in Instruction Register Program Counter increments to point to next instruction Step 2 — DECODE: Control Unit reads the instruction in IR Understands what operation is needed and what data is required Step 3 — EXECUTE: ALU performs the operation Result stored in Accumulator or written back to memory ↺ Repeat millions/billions of times per second
✅ Conclusion
The CPU is the heart of every computer. ALU handles calculations, Control Unit manages operations, and Registers provide ultra-fast temporary storage. Together through the Fetch-Execute Cycle, they execute any program.
💡
Memory Tip

CPU = ALU (calculations) + CU (coordinator) + Registers (fast notes). Cycle = Fetch → Decode → Execute → Repeat. Like reading a recipe (Fetch), understanding it (Decode), then cooking (Execute)!

What is the OSI Model? Explain all 7 layers with their specific functions.
🔴 ★★★★★ EVERY YEAR 📅 2019 · 2020 · 2021 · 2022 · 2023 · 2024 · 2025 — ALL 7 years!
▼
✍ Write Exactly Like This
📌 Introduction
OSI stands for Open Systems Interconnection. It is a standard model by ISO that divides network communication into 7 separate layers. Each layer has a specific function, and together they ensure data is sent and received correctly between any two computers.
🗂️ The 7 Layers (Top to Bottom)
Layer 7 — APPLICATION: User interface · HTTP · SMTP · FTP Layer 6 — PRESENTATION: Data format · Encryption (SSL) · Compression Layer 5 — SESSION: Opens/manages/closes communication sessions Layer 4 — TRANSPORT: Reliable delivery · Segments · TCP / UDP Layer 3 — NETWORK: Routing · IP Addresses · Router device Layer 2 — DATA LINK: MAC Addressing · Frames · Switch device Layer 1 — PHYSICAL: Cables · Hubs · Raw bits (0s and 1s)
🧠 Memory Phrase: "All People Seem To Need Data Processing"
A=Application · P=Presentation · S=Session · T=Transport · N=Network · D=Data Link · P=Physical
✅ Conclusion
The OSI model provides a universal framework for network communication. Each layer serves a specific purpose and interacts only with the layers directly above and below it.
💡

"All People Seem To Need Data Processing" → Application, Presentation, Session, Transport, Network, Data Link, Physical. Learn THIS phrase and you'll never forget!

D1
DERIV
What are Stationary Waves? Derive frequency formula for nth harmonic in a stretched string.
🔴 ★★★★★ 7 SOURCES! 📅 DJ Prelim + Adamjee + BIEK Model + 2025 + 2024 + 2023 + 2022
▼
✍ Write Exactly Like This
📌 Definition
A Stationary Wave is formed when two identical waves — same frequency, wavelength, and amplitude — travel in opposite directions along the same medium and superpose. The result appears stationary (not moving forward).
Node: Point where displacement is always ZERO (no vibration)
Antinode: Point where displacement is MAXIMUM (greatest vibration)
📐 Derivation
String of length L, fixed at both ends. Tension = T. Mass per unit length = μ. Wave speed: v = √(T/μ)
1st HARMONIC (Fundamental): Only half a wavelength fits → L = λ₁/2 → λ₁ = 2L f₁ = v/λ₁ = v/2L = (1/2L) × √(T/μ) 2nd HARMONIC: One full wavelength → L = λ₂ → f₂ = v/L = 2f₁ f₂ = (2/2L) × √(T/μ) 3rd HARMONIC: L = 3λ₃/2 → f₃ = 3v/2L = 3f₁ f₃ = (3/2L) × √(T/μ) ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ nth HARMONIC (GENERAL FORMULA): fₙ = (n/2L) × √(T/μ) where n = 1, 2, 3, ... ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✅ Key Conclusions
All harmonics are whole-number multiples of the fundamental frequency
Frequency INCREASES when: Tension ↑ OR Length ↓ OR Mass per unit length ↓
Guitar string produces higher pitch when tightened (T increases)
💡
Memory Tip

Formula: fₙ = (n/2L)√(T/μ). First harmonic = HALF a wave in the string. More tension = higher frequency (tight guitar string = high pitch). Node = no movement, Antinode = max movement.

D2
DERIV
Derive velocity expressions for two objects after a perfectly ELASTIC head-on collision.
🔴 ★★★★★ 6 SOURCES 📅 DJ + Adamjee + BIEK Model + 2025 + 2024 + 2023
▼
✍ Write Exactly Like This
📌 Definition
Elastic Collision: A collision where BOTH momentum AND kinetic energy are completely conserved. No energy lost to heat or deformation. Objects bounce off perfectly.
📐 Derivation — 4 Steps
SETUP: masses m₁ and m₂ Before collision: m₁ at velocity u₁, m₂ at velocity u₂ After collision: m₁ at velocity v₁, m₂ at velocity v₂ STEP 1 — Conservation of Momentum: m₁u₁ + m₂u₂ = m₁v₁ + m₂v₂ → m₁(u₁ - v₁) = m₂(v₂ - u₂) ...(1) STEP 2 — Conservation of Kinetic Energy: ½m₁u₁² + ½m₂u₂² = ½m₁v₁² + ½m₂v₂² → m₁(u₁² - v₁²) = m₂(v₂² - u₂²) Using a²-b² = (a+b)(a-b): → m₁(u₁+v₁)(u₁-v₁) = m₂(v₂+u₂)(v₂-u₂) ...(2) STEP 3 — Divide equation (2) by equation (1): → u₁ + v₁ = v₂ + u₂ ★ KEY RESULT: u₁ - u₂ = v₂ - v₁ (Velocity of approach = Velocity of separation) STEP 4 — Solve for v₁ and v₂: ┌─────────────────────────────────────────────┐ │ v₁ = [(m₁-m₂)u₁ + 2m₂u₂] / (m₁+m₂) │ │ v₂ = [(m₂-m₁)u₂ + 2m₁u₁] / (m₁+m₂) │ └─────────────────────────────────────────────┘ SPECIAL CASE — Equal masses (m₁ = m₂): v₁ = u₂ and v₂ = u₁ (they EXCHANGE velocities!) Like billiard balls!
💡
Memory Tip

Two conservation laws: (1) Momentum (2) KE. Divide law2 by law1 → KEY STEP: velocity of approach = velocity of separation. Equal masses = swap velocities like billiard balls!

D3
DERIV
Derive the expression for electric field at a point on the EQUATORIAL line of an electric dipole.
🟡 ★★★★ 5 SOURCES 📅 DJ + Adamjee + BIEK Model + 2024 + 2023
▼
✍ Write Exactly Like This
📌 Definition
Electric Dipole: Two equal and opposite charges +q and −q separated by distance 2a. Dipole moment: p = q × 2a (direction: from −q to +q)
📐 Derivation
Point P on equatorial line at distance y from center O Distance from each charge to P: r = √(y² + a²) STEP 1 — Field due to +q at P: E₊ = kq/(y² + a²) [away from +q] STEP 2 — Field due to −q at P: E₋ = kq/(y² + a²) [toward −q] Note: |E₊| = |E₋| (equidistant) STEP 3 — Resolve components: ✗ VERTICAL components → CANCEL each other ✓ HORIZONTAL components → ADD together STEP 4 — Net field: E = 2E₊cosθ where cosθ = a/√(y² + a²) E = 2 × [kq/(y²+a²)] × [a/√(y²+a²)] E = 2kqa / (y²+a²)^(3/2) Since p = 2qa: ┌─────────────────────────────────────┐ │ E = kp / (y² + a²)^(3/2) │ │ For short dipole (y >> a): E ≈ kp/y³ │ └─────────────────────────────────────┘ DIRECTION: OPPOSITE to dipole moment vector!
💡
Memory Tip

At equatorial point: VERTICAL components CANCEL, HORIZONTAL components ADD. Direction is always OPPOSITE to dipole moment p. Formula: E = kp/(y²+a²)^(3/2)

DERIV
What are Newton's Rings? How are they formed? Derive radius of nth bright ring.
🔴 ★★★★★ 7 CONSECUTIVE YEARS! 📅 2019 · 2020 · 2021 · 2022 · 2023 · 2024 · 2025 — NEVER MISSED!
▼
✍ Write Exactly Like This
📌 Definition + Formation
Newton's Rings are circular, alternating bright and dark interference fringes formed when monochromatic light falls on a plano-convex lens placed on a flat glass plate.
Formation: A thin air film is trapped between the curved lens surface and the flat plate. The film thickness is ZERO at the center (touching point) and increases outward. Reflected light from the two surfaces interferes — constructive = bright rings, destructive = dark rings.
⚠️ CENTER IS ALWAYS DARK — because air gap is zero at center, causing half-wavelength phase shift on reflection!
📐 Derivation of nth Bright Ring Radius
STEP 1 — Air gap thickness at distance r from center: t = r²/(2R) [from geometry of circle] STEP 2 — Path difference for reflected light: Path difference = 2t + λ/2 (λ/2 due to phase change at glass plate reflection) STEP 3 — Condition for nth BRIGHT ring: 2t + λ/2 = (2n−1)λ/2 → 2t = (2n−1)λ/2 − λ/2 ... (simplifying) STEP 4 — Substitute t = r²/(2R): r²/R = (2n−1)λ/2 r² = (2n−1)λR/2 ┌────────────────────────────────────────────┐ │ Bright ring: rₙ = √[(2n−1)λR/2] │ │ Dark ring: rₙ = √(nλR) │ │ where R = radius of curvature of lens │ └────────────────────────────────────────────┘
💡
Memory Tip — SAFEST question in Physics!

CENTER IS ALWAYS DARK. Bright ring: rₙ = √[(2n-1)λR/2]. Dark ring: rₙ = √(nλR). Appeared in 7 consecutive years — this is your guaranteed 10 marks if you memorize it!

D5
DERIV
Define escape velocity. Derive its formula and calculate its value for Earth.
🟡 ★★★★ HIGH CHANCE 📅 2024SG · 2022 · 2020 · 2018 — pattern shows 2026!
▼
✍ Write Exactly Like This
📌 Definition
Escape velocity is the minimum velocity given to an object at the surface of a planet so it can overcome gravitational pull and travel to infinity without any further propulsion.
📐 Derivation
STEP 1 — Energy at Earth's surface: KE = ½mvₑ² PE = −GMm/R (negative: gravity is attractive) Total Energy = ½mvₑ² − GMm/R STEP 2 — Energy at infinity: KE = 0 (velocity = 0 at infinity) PE = 0 (no gravity at infinity) Total Energy = 0 STEP 3 — Conservation of Energy: ½mvₑ² − GMm/R = 0 ½mvₑ² = GMm/R vₑ² = 2GM/R ┌─────────────────────────────┐ │ vₑ = √(2GM/R) │ └─────────────────────────────┘ STEP 4 — Simplify using g = GM/R²: GM = gR² vₑ = √(2gR) STEP 5 — Calculate for Earth: g = 9.8 m/s², R = 6.4 × 10⁶ m vₑ = √(2 × 9.8 × 6.4 × 10⁶) vₑ = √(125.44 × 10⁶) ┌────────────────────────────────────┐ │ vₑ ≈ 11,200 m/s = 11.2 km/s │ └────────────────────────────────────┘
💡
Memory Tip

Formula: vₑ = √(2gR) = 11.2 km/s for Earth. Logic: KE at surface must equal PE so they cancel and object reaches infinity with zero velocity remaining. Moon's escape velocity = only 2.4 km/s (less mass).

What is Doppler Effect? If ambulance moves toward you, what happens to pitch?
🔴 ★★★★★2024 · 2023 · 2022 + numerical every year
▼
✍ Answer
Doppler Effect: The apparent change in frequency of a wave due to relative motion between the wave source and observer.
Formula: f' = f × (v + v_L)/(v − v_S) where: v = speed of sound, v_L = listener speed, v_S = source speed WHEN SOURCE MOVES TOWARD OBSERVER: Sound waves get COMPRESSED → wavelength decreases → Frequency (pitch) INCREASES → Higher pitch heard WHEN SOURCE MOVES AWAY FROM OBSERVER: Sound waves get STRETCHED → wavelength increases → Frequency (pitch) DECREASES → Lower pitch heard REAL EXAMPLE: Ambulance siren sounds HIGHER as it approaches and LOWER as it moves away.

BALANCE CONDITION: When no current flows through galvanometer: ┌──────────────────────────────┐ │ P/Q = R/S OR P×S = Q×R │ └──────────────────────────────┘ To find unknown resistance R: R = (P × S) / Q Application: Very accurate — used in electronic measuring instruments. ⚠️ TIP: Draw the diamond shape in exam for full marks!

Computer Science — Key Points

CPU Components: ALU · Control Unit · Registers · Cache. Fetch → Decode → Execute cycle.
OSI 7 Layers: Physical · Data Link · Network · Transport · Session · Presentation · Application
Networks: LAN <1km · MAN ~100km · WAN worldwide. Bus/Ring/Star/Mesh topologies.
RAM vs ROM: RAM = volatile, temporary. ROM = non-volatile, permanent (BIOS).
Bus Types: Data Bus (data), Address Bus (location), Control Bus (signals). Width = bits at once.
IP Classes: A: 0–127 · B: 128–191 · C: 192–223. Subnet mask separates network/host.
⚛️

Physics — Key Formulas

Escape Velocity: v = √(2GM/R) ≈ 11.2 km/s. Minimum speed to leave Earth's gravity.
Elastic Collision: KE conserved. m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂'. Both momentum & KE.
Dipole Moment: p = q·d (C·m). Direction: −ve to +ve charge. Torque τ = p×E.
Stationary Waves: Two identical waves, opposite directions. Nodes = zero displacement. Antinodes = max.
Newton's Rings: r² = nλR. Circular interference fringes. Radius² ∝ ring number.
SHM: a = −ω²x. T = 2π√(l/g) pendulum. F = −kx (Hooke's Law).
📐

Mathematics — Key Formulas

Double Angle: sin2θ = 2sinθcosθ. cos2θ = cos²θ−sin²θ = 2cos²θ−1 = 1−2sin²θ.
Gauss-Jordan: Augmented matrix [A|b] → row ops → RREF → read solution directly.
Induction Steps: 1) Base n=1 2) Assume n=k 3) Prove n=k+1 4) Conclude ∀n∈ℕ.
Linear Programming: Graph constraints → find feasible region → test corner points → max/min at vertex.
Inverse Function: f⁻¹ exists iff f is 1-to-1. Swap x,y then solve for y. Range of f = Domain of f⁻¹.
Binomial: (a+b)ⁿ = Σ C(n,r)·aⁿ⁻ʳ·bʳ. General term = C(n,r)·aⁿ⁻ʳ·bʳ.
